# 🧠 AI Friend Finder

A personality-based friend matching web app using NLP + MBTI/Big Five traits.

Built with FastAPI, React, and HuggingFace.

## Structure
- `apps/frontend`: Vite + React UI
- `apps/backend`: FastAPI server
- `packages/personality-model`: NLP-based personality detection
- `infra`: Deployment configs (Render, Supabase, etc.)

## Scripts
- `npm run dev` – Start all dev servers
- `npm run build` – Build all apps
